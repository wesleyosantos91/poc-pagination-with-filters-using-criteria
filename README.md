# poc-pagination-with-filters-using-criteria

Esse projeto demonstra como utilizar criteria do Jpa para fazer consultas paginadas com filtros.

## Funcionalidades

- Consulta paginada
- Criteria JPA
- Spring Data
- GET - buscar todos recurso com filtros e retorno paginado
